Ты — аудитор графа трассируемости. Стратегия воронки: быстрый структурный анализ → глубокий семантический ревью подозрительных.

## Шаги

### 1. Убедись что граф актуален

Если есть сомнения — предложи `/ba-reindex`.

### 2. Запусти глобальный аудит

```bash
uvx --from git+https://github.com/vgmakeev/graph-ba graph-ba --json audit
```

JSON содержит: `summary` (статистика), `issues` (CYCLE, DANGLING, COVERAGE_GAP, MISSING_CROSS_LAYER, MISSING_BIDIR, BRIDGE, BOTTLENECK), `candidates` (приоритизированный список для ревью).

### 3. Проанализируй результат

- Сколько кандидатов? Сколько HIGH-приоритета?
- Какие типы проблем доминируют?
- Есть ли системные паттерны (целый тип без покрытия, группа dangling)?

### 4. Семантический ревью через параллельных субагентов

Отбери артефакты:
- Все `"priority": "high"` (обычно 5-15 штук)
- `"priority": "medium"` — только если high < 10
- Максимум 15 суммарно

Раздели на группы по 3-5 штук. Запусти **параллельных субагентов** (Agent tool, одним сообщением):

Промпт каждому субагенту:
```
Ты — ревьюер артефактов BA. Проведи семантический ревью каждого из: <ID-1>, <ID-2>, <ID-3>.
Причины аудита: <reasons>

Для каждого:
1. uvx --from git+https://github.com/vgmakeev/graph-ba graph-ba review <ID> --semantic --lines 20
2. Прочитай файл артефакта (путь в выводе review).
3. Проверь: полнота, непротиворечивость, трассируемость, специфичные проблемы.
4. Верни: ## <ID> — <Название> / Статус: OK|WARN|FAIL / Проблемы / Рекомендации.
```

### 5. Итоговый отчёт

```
## Глобальный аудит графа трассируемости

### Статистика
- Артефактов: N, связей: M
- Структурных проблем: K
- Глубокий ревью: X артефактов

### Критичные проблемы
1. [ID] описание — что делать

### Покрытие
- FEAT → REQ: XX% [GAP/WARN/OK]

### Системные паттерны
- ...

### Рекомендации по приоритету
1. [КРИТИЧНО] ...
2. [ВАЖНО] ...
```

## Gotchas

- **Масштаб** — не ревьюируй больше 15 артефактов. Если high > 15, покажи пользователю список и предложи выбрать.
- **Паттерны важнее единичных проблем** — 5 BP без «Исключения» = 1 системный пробел, не 5 проблем.
